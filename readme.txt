== Description ==
This plugin made for display image hover effect. You can make your hover grid image using Visual composer page builder fast and easy.

Main Features:

* Responsive Design
* Easy-to-use with Shortcode Generator Using Visual Composer
* 34 Awesome Hover Effects
* Grid Layout
* Set Overlay Background Color
* Set Gutter Space Between Columns
* Set Title And Description Align
* Set Link For Images
* Add Custom CSS And Custom Class For Image To Make Your Custom View
* Supports all browsers (including IE7+ ,Chrome, Safari or Firefox)

== Installation ==

Installing from WordPress
Follow the steps below to install inside Wordpress
* Navigate to Appearance -> Plugins
* Click Install Plugins and then Upload
* Navigate to iThemeland-Image-Hover-Effect-For-VC-wp.zip on your computer and click Install Now
* To activate the newly installed plugin navigate to Appearance -> Plugins select the iThemeland Image Hover Effect for VC - wp  and press Activate
* After install, activate the plugin.


Installing via FTP Server
Follow the steps below to install via FTP
* Unzip iThemeland-Image-Hover-Effect-For-VC-wp.zip to a desired location on your HDD. These extracted files will be uploaded later via FTP
* Connect to your server via FTP
* Navigate to /wp-content/plugins/ on server
* Navigate to location of unzipped iThemeland-Image-Hover-Effect-For-VC-wp.zip
* Upload local folder /iThemeland-Image-Hover-Effect-For-VC-wp / to remote /wp-content/plugins/
* To activate the newly installed plugin navigate to Appearance -> Plugins select the iThemeland Image Hover Effect for VC - wp  and press Activate
* After install, activate the plugin.
